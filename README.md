# 🎤 Text to Speech Converter

A simple, elegant web application that converts written text into spoken words using the Web Speech API. Just type or paste your text, click a button, and listen to your content being read aloud.

## ✨ Features

- **Instant Text-to-Speech Conversion** - Convert any text to speech with one click
- **Clean, Modern UI** - Eye-catching gradient background and intuitive interface
- **Error Handling** - User-friendly error messages when no text is provided
- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Visual Feedback** - Button text changes to indicate when speech is playing
- **Cross-Browser Compatible** - Works on all modern browsers that support the Web Speech API


## 🛠️ Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling and animations (gradients, hover effects, responsive design)
- **JavaScript (ES6)** - Core functionality and Web Speech API integration
- **Web Speech API** - Browser's native speech synthesis
